## 1. System Overview
EAOS is an AI-native Enterprise Architecture Operating System that transforms 
enterprise architecture from static documentation into executable software.

## 2. Layering & Boundaries
- **Core Kernel:** Governance, Federation (Raft/BFT), Splay RAM Cache.
- **Platform Services:** Security, Telemetry (OTLP), Resilience, Performance.
- **Packages:** Bounded Contexts for Knowledge, Identity, Intelligence, Tenancy.
- **Delivery Channels:** REST API Gateway, CLI, Web Control Room.
